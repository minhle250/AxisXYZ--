Axis XYZ-Sigma is a high-performance, strategic operating system interface designed for advanced intel synthesis, tactical task management, and visual simulation. Built with React, Vite, and Tailwind CSS, it provides a seamless command center experience.

## 🚀 Features

- **Hegemony Hub**: Tactical campaign management and weaponized intel tracking.
- **Private Library**: Strategic intel synthesis and analysis tools.
- **Visual Simulator**: High-fidelity visual synthesis powered by Gemini 1.5 Flash.
- **Dispatcher**: Integrated Pomodoro-style Deep Work Lock and Tactical Task management.
- **Liquid Glass UI**: A distinctive, high-contrast interface designed for focus and clarity.

## 🛠️ Setup & Deployment

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/AxisXYZ--.git
   cd AxizXYZ--
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Copy the example environment file and fill in your credentials:
   ```bash
   cp .env.example .env.local
   ```
   Edit `.env.local` and provide your `VITE_GEMINI_API_KEY` and Firebase configuration if applicable.

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

### Building for Production

To create a production-ready build:
```bash
npm run build
```
The optimized assets will be generated in the `dist/` directory.

## 🛡️ Security & Protocols

- **Environment Variables**: All sensitive keys are managed via `.env` files and never committed to version control.
- **Iframe Isolation**: Integrated Google apps are handled with secure iframe protocols and direct-launch fallbacks.
- **Persistence**: Tactical data (notes, tasks) is persisted locally via `localStorage` for privacy and speed.
